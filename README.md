# SQL Practice Projects

Welcome to my SQL Practice Projects repository! Here you will find a collection of SQL exercises, projects, and a dashboard that I've created to enhance my skills in SQL and data analysis.


## About

This repository contains various SQL projects that I have worked on to improve my database querying and data manipulation skills. Each project includes SQL scripts, data sets, and documentation to help you understand the work done.

## Projects

Here are some of the SQL projects included in this repository:

1. **pizzahut Sales**
   - **Description**: A project to manage employee records using SQL queries to insert, update, delete, and retrieve employee data.
   - **Key Queries**: `SELECT`, `JOIN`, `GROUP BY`, `ORDER BY`

2. **Sales Data Analysis**
   - **Description**: Analysis of sales data to understand trends and patterns using complex SQL queries.
   - **Key Queries**: `INNER JOIN`, `OUTER JOIN`, `SUBQUERIES`, `AGGREGATE FUNCTIONS`

3. **Library Management System**
   - **Description**: A SQL project focused on managing a library's book inventory and user transactions.
   - **Key Queries**: `CREATE TABLE`, `INSERT`, `UPDATE`, `DELETE`

4. **Customer Relationship Management (CRM)**
   - **Description**: A CRM database project for tracking customer interactions and sales.
   - **Key Queries**: `UNION`, `CASE`, `CROSS
